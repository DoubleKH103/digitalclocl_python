# Digital Clock – PyQt5 ⏰

A simple **digital clock desktop application** built using **PyQt5**.  
The clock updates every second and displays the current system time in a clean, minimal UI.

## Preview
The application shows:
- Hours, minutes, and seconds
- AM / PM format
- Green digital-style text on a black background

## Features
- Real-time clock updated every second
- Uses `QTimer` for continuous updates
- Simple and clean UI
- Centered digital time display
- Lightweight PyQt5 application
